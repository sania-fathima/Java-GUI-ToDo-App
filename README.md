# Smart To-Do Manager

## Overview

Smart To-Do Manager is a Java Swing-based desktop application that helps users organize, prioritize, and track daily tasks. The application provides a modern dark-themed interface with support for task categories, priorities, due dates, completion tracking, searching, and automatic data persistence.

## Features

### Task Management

* Add Tasks
* Delete Tasks
* Mark Tasks as Completed
* Clear All Tasks
* Search Tasks

### Organization

* Categories (Study, Work, Personal, Health)
* Priority Levels (High, Medium, Low)
* Due Dates

### Dashboard

* Total Tasks Counter
* Completed Tasks Counter
* Pending Tasks Counter
* Status Notifications

### Data Persistence

* Save Tasks Automatically
* Load Tasks Automatically on Startup
